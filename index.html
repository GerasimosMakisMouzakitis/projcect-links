Your Projects

How to use:

  1. Create a new project by entering a name and clicking "Create Project"
  2. Select a project by clicking on it (it will be highlighted)
  3. To add links: Drag URLs from your browser's address bar, bookmarks, or links from web pages directly into the drop zone
  4. Alternatively, click the drop zone to add links manually
  5. Use the export button to download your project links

ðŸ’¡ Tips for drag and drop:

  • Drag from the browser's address bar (URL bar) for best results
  • Drag bookmarks directly from your bookmarks bar
  • Drag links from web pages (right-click and drag)
  • Make sure to select a project first!
